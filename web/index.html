Onionoo/TorStatusProtocol REST API

Summary documents

Summary list of relays with nicknames, fingerprints, IP addresses, and running information and/or bridges with hashed fingerprints and running information.

GET summary/all Return the list of all relays and bridges that are currently running or that have been running in the past week.
GET summary/running Return the list of running relays and bridges.
GET summary/relays Return only the list of relays.
GET summary/bridges Return only the list of bridges.
GET summary/search/:searchtext Return the list of relays and bridges with searchtext matching nickname, fingerprint, IP address, or hashed fingerprint.
GET summary/lookup/:fingerprint Return the relay or bridge with the given fingerprint or hashed fingerprint.

Detail documents

Detail information for one or more relays or bridges.

GET details/all Return detail documents for all relays and bridges.
GET details/running Return detail documents for running relays and bridges.
GET details/relays Return detail documents for all relays.
GET details/bridges Return detail documents for all bridges.
GET details/search/:searchtext Return detail documents for relays and bridges with searchtext matching nickname, fingerprint, IP address, or hashed fingerprint.
GET details/lookup/:fingerprint Return the detail document for the given relay or bridge only.

Bandwidth documents

Bandwidth information for one or more relays or bridges.

GET bandwidth/all Return bandwidth documents for all relays and bridges.
GET bandwidth/running Return bandwidth documents for running relays and bridges.
GET bandwidth/relays Return bandwidth documents for all relays.
GET bandwidth/bridges Return bandwidth documents for all bridges.
GET bandwidth/search/:searchtext Return bandwidth documents for relays and bridges with searchtext matching nickname, fingerprint, IP address, or hashed fingerprint.
GET bandwidth/lookup/:fingerprint Return the bandwidth document for the given relay or bridge only.
